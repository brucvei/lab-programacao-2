# Caça Palavras

### Arquivos
- main.c -> arquivo principal
- main.h -> arquivo de cabeçalho e declaracao de funcoes

### Implementado

- Lê o tamanho do grid (altura e largura da matriz) e a frase que preenche o grid
- Lê a palavra a ser procurada ou opção de saída, regras, palavras encontradas e terminar o jogo
- Verifica se a palavra a ser procurada existe no grid na horizontal (esquerda -> direita) e vertical (cima -> baixo)

### Não Implementado

- Horizontal (direita -> esquerda)
- Vertical (baixo -> cima)
- Diagonal (qualquer direcao)
